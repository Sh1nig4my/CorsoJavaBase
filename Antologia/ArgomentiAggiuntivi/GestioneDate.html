Lezione Aggiuntiva: Le Date in Java

Introduzione

In questa lezione aggiuntiva ci focalizzeremo sull'uso delle date in Java, un argomento fondamentale per la gestione del tempo e delle scadenze all'interno delle applicazioni. Esploreremo le principali classi e metodi forniti dal package java.time, introdotto con Java 8.

Gestione delle Date in Java (Java 8+)

Con Java 8 è stata introdotta la libreria java.time, che rappresenta un grande miglioramento rispetto alle classi Date e Calendar. Le nuove API forniscono una gestione più intuitiva e moderna di date, orari e fusi orari. Le principali classi come LocalDate, LocalTime, LocalDateTime e ZonedDateTime sono immutabili e thread-safe, progettate per essere facili da usare e meno soggette ad errori.

Di seguito esploreremo alcune delle classi più utilizzate nella libreria java.time e i loro casi d'uso più comuni.

2.1 LocalDate

La classe LocalDate rappresenta una data senza considerare l'ora del giorno. È ideale per lavorare con calendari, compleanni e altre applicazioni legate solo alla data.

Esempio di utilizzo:

import java.time.LocalDate;
        
        public class DateExample {
            public static void main(String[] args) {
                // Ottenere la data odierna
                LocalDate today = LocalDate.now();
        
                // Creare una data specifica
                LocalDate specificDate = LocalDate.of(2023, 1, 1);
        
                // Confronto tra date
                boolean isBefore = specificDate.isBefore(today);
                boolean isAfter = specificDate.isAfter(today);
        
                System.out.println("Data di oggi: " + today);
                System.out.println("Data specifica: " + specificDate);
                System.out.println("La data specifica è precedente alla data odierna? " + isBefore);
                System.out.println("La data specifica è successiva alla data odierna? " + isAfter);
            }
        }

2.2 LocalTime

La classe LocalTime rappresenta un orario del giorno senza una data associata. È utile per gestire orari come quelli di apertura/chiusura di attività o programmazioni orarie.

Esempio di utilizzo:

import java.time.LocalTime;
        
        public class TimeExample {
            public static void main(String[] args) {
                // Ottenere l'orario attuale
                LocalTime now = LocalTime.now();
        
                // Creare un orario specifico
                LocalTime specificTime = LocalTime.of(14, 30);
        
                // Aggiungere o sottrarre tempo
                LocalTime timePlusHours = specificTime.plusHours(2);
                LocalTime timeMinusMinutes = specificTime.minusMinutes(15);
        
                System.out.println("Orario attuale: " + now);
                System.out.println("Orario specifico: " + specificTime);
                System.out.println("Orario specifico più 2 ore: " + timePlusHours);
                System.out.println("Orario specifico meno 15 minuti: " + timeMinusMinutes);
            }
        }

2.3 LocalDateTime

La classe LocalDateTime combina una data e un'ora in un singolo oggetto. È utile per registrare timestamp senza tener conto del fuso orario.

Esempio di utilizzo:

import java.time.LocalDateTime;
        
        public class DateTimeExample {
            public static void main(String[] args) {
                // Ottenere la data e l'ora attuale
                LocalDateTime now = LocalDateTime.now();
        
                // Creare una data e un orario specifici
                LocalDateTime specificDateTime = LocalDateTime.of(2023, 1, 1, 14, 30);
        
                // Aggiungere giorni e ore
                LocalDateTime futureDateTime = specificDateTime.plusDays(5).plusHours(3);
        
                System.out.println("Data e ora attuale: " + now);
                System.out.println("Data e ora specifica: " + specificDateTime);
                System.out.println("Data e ora futura: " + futureDateTime);
            }
        }

2.4 Manipolazione delle Date

Le API Java 8 permettono una manipolazione facile ed elegante di date e orari. Puoi aggiungere o sottrarre giorni, mesi o anni in modo semplice utilizzando metodi come plusDays() o minusMonths().

Esempio di manipolazione delle date:

import java.time.LocalDate;
        
        public class ManipulateDate {
            public static void main(String[] args) {
                LocalDate today = LocalDate.now();
        
                // Aggiungere giorni e mesi
                LocalDate nextWeek = today.plusWeeks(1);
                LocalDate lastMonth = today.minusMonths(1);
        
                // Modificare l'anno
                LocalDate nextYear = today.plusYears(1);
        
                System.out.println("Data di oggi: " + today);
                System.out.println("Data della prossima settimana: " + nextWeek);
                System.out.println("Data del mese scorso: " + lastMonth);
                System.out.println("Data del prossimo anno: " + nextYear);
            }
        }

2.5 ZonedDateTime e Fusi Orari

La classe ZonedDateTime aggiunge il supporto per i fusi orari, rendendola ideale per applicazioni globali che necessitano di gestire il tempo in contesti internazionali.

Esempio di utilizzo:

import java.time.ZonedDateTime;
        import java.time.ZoneId;
        
        public class ZonedDateTimeExample {
            public static void main(String[] args) {
                // Ottenere la data e l'ora attuale in un fuso orario specifico
                ZonedDateTime nowInUTC = ZonedDateTime.now(ZoneId.of("UTC"));
                ZonedDateTime nowInNewYork = ZonedDateTime.now(ZoneId.of("America/New_York"));
        
                System.out.println("Data e ora attuale in UTC: " + nowInUTC);
                System.out.println("Data e ora attuale a New York: " + nowInNewYork);
            }
        }
