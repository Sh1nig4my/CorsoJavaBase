Lezione 26: Spring Boot, API REST e Tipi di Richieste HTTP

1. Introduzione

Spring Boot è un framework che semplifica la creazione di applicazioni Java basate su Spring. Oggi esploreremo il concetto di API REST e i vari tipi di richieste HTTP utilizzate per comunicare con un server backend.

2. Concetti Chiave

Un'API REST (Representational State Transfer) è un'architettura che consente la comunicazione tra client e server utilizzando il protocollo HTTP.

Le principali operazioni supportate sono:

  • GET: Recupera informazioni da una risorsa.
  • POST: Crea una nuova risorsa.
  • PUT: Aggiorna una risorsa esistente.
  • DELETE: Elimina una risorsa.

Esempio di API REST con Spring Boot

import org.springframework.web.bind.annotation.*;
import java.util.*;

@RestController
@RequestMapping("/api")
public class ExampleController {
    private List items = new ArrayList<>();

    @GetMapping("/items")
    public List getItems() {
        return items;
    }

    @PostMapping("/items")
    public void addItem(@RequestBody String item) {
        items.add(item);
    }

    @PutMapping("/items/{index}")
    public void updateItem(@PathVariable int index, @RequestBody String newItem) {
        items.set(index, newItem);
    }

    @DeleteMapping("/items/{index}")
    public void deleteItem(@PathVariable int index) {
        items.remove(index);
    }
}
