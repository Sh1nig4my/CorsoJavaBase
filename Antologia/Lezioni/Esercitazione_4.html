Esercitazione: Gioco Testuale

1. Introduzione

Questa esercitazione consiste nella realizzazione di un semplice gioco testuale in Java. L'obiettivo è applicare le conoscenze acquisite su oggetti, collezioni, input/output e gestione delle eccezioni per creare un'ambientazione interattiva.

2. Struttura del Gioco

Il gioco si sviluppa attraverso le seguenti componenti principali:

  • Classe `Gioco`: gestisce il flusso del gioco e l'interazione con l'utente.
  • Classe `Stanza`: rappresenta un ambiente del gioco con descrizione e possibili direzioni.
  • Classe `Giocatore`: contiene le informazioni del giocatore come inventario e posizione attuale.
  • Classe `Oggetto`: rappresenta gli oggetti che il giocatore può raccogliere.

3. Esempio di Implementazione

import java.util.HashMap;
import java.util.Scanner;

class Stanza {
    private String descrizione;
    private HashMap uscite = new HashMap<>();

    public Stanza(String descrizione) {
        this.descrizione = descrizione;
    }

    public void impostaUscita(String direzione, Stanza stanza) {
        uscite.put(direzione, stanza);
    }

    public Stanza vai(String direzione) {
        return uscite.get(direzione);
    }

    public String getDescrizione() {
        return descrizione;
    }
}

public class Gioco {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        Stanza ingresso = new Stanza("Sei all'ingresso di un castello misterioso.");
        Stanza salone = new Stanza("Ti trovi in un grande salone con un camino acceso.");
        ingresso.impostaUscita("nord", salone);

        Stanza stanzaCorrente = ingresso;
        while (true) {
            System.out.println(stanzaCorrente.getDescrizione());
            System.out.print("Dove vuoi andare? ");
            String direzione = scanner.nextLine();

            if (stanzaCorrente.vai(direzione) != null) {
                stanzaCorrente = stanzaCorrente.vai(direzione);
            } else {
                System.out.println("Non puoi andare in quella direzione.");
            }
        }
    }
}

4. Esercizio Pratico

Espandi il gioco aggiungendo:

  • Un inventario per il giocatore.
  • Oggetti collezionabili nelle stanze.
  • Azioni interattive con gli oggetti.
