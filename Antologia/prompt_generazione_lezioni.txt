Prompt per Generare una Lezione in Formato HTML per il Corso di Programmazione Backend

Devo generare una lezione per il mio corso di programmazione backend in formato HTML, seguendo lo stile e la struttura delle lezioni già create. Il corso copre argomenti relativi a Java e al framework Spring.

Requisiti per la Lezione
Formato HTML: La lezione deve essere un file HTML strutturato come le lezioni precedenti, con titoli, paragrafi, esempi di codice, sezioni di test, approfondimenti ed esercizi.
Stile CSS: Deve utilizzare lo stesso stile CSS delle lezioni già create (file style.css).
Struttura della Lezione:
Titolo della lezione: [Titolo della lezione basato sul programma del corso]
Sezione 1: Introduzione: Una spiegazione introduttiva dell'argomento.
Sezione 2: Concetti principali: Approfondimento delle teorie fondamentali con codice di esempio.
Sezione 3: Esempi pratici: Codice Java funzionante e spiegazioni.
Sezione 4: Test e domande: Una serie di domande per verificare la comprensione dello studente.
Sezione 5: Esercizi pratici: Compiti per esercitarsi sugli argomenti trattati.
Sezione 6: Bibliografia: Link alla documentazione ufficiale e a risorse di approfondimento.
Sezione 7: Avanzata: Approfondimenti tecnici per studenti più esperti.
Contenuto della Lezione
La lezione da generare deve coprire il seguente argomento, come specificato nel programma del corso (file index.html):

Giorno [numero]: [Nome e argomenti del giorno]
Stile e Linguaggio
Il tono deve essere formativo e professionale, simile a un manuale di programmazione.
Ogni sezione deve essere ben spiegata e accompagnata da esempi di codice funzionante in Java.
Test e domande devono verificare la comprensione degli studenti.
Esercizi pratici devono essere progettati per mettere in pratica quanto appreso.
La sezione avanzata deve fornire spunti di studio per approfondire l'argomento.
Formato del Codice
Il codice deve essere completo, funzionante e commentato, per essere facilmente comprensibile agli studenti.
Il codice Java deve essere formattato all'interno dei tag <pre><code> per una corretta visualizzazione.

Il file HTML deve essere simile a quelli già generati (es. Lezione_1.html), e seguire questa struttura:
