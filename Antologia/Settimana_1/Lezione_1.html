Lezione 1: Introduzione alla Programmazione

1. Concetti fondamentali di Programmazione

La programmazione è l'arte di scrivere istruzioni che un computer può comprendere e eseguire. Comprendere i concetti di base è essenziale per costruire una solida base di conoscenza.

Un algoritmo è una sequenza logica di passi per risolvere un problema. Può essere rappresentato tramite pseudocodice o diagrammi di flusso, ed è il cuore della programmazione.

Esempio di Algoritmo:

Pseudocodice per calcolare la somma di due numeri in Java:

public class SommaNumeri {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
    
            // Leggi il primo numero
            System.out.print("Inserisci il primo numero: ");
            int numero1 = scanner.nextInt();
    
            // Leggi il secondo numero
            System.out.print("Inserisci il secondo numero: ");
            int numero2 = scanner.nextInt();
    
            // Calcola la somma
            int somma = numero1 + numero2;
    
            // Stampa il risultato
            System.out.println("La somma è: " + somma);
        }
}

Un altro concetto chiave sono le strutture dati, che aiutano a organizzare e gestire informazioni. Alcuni esempi includono array, liste, stack e queue.

Esempio: Array

Un esempio di utilizzo degli array in Java:

public class EsempioArray {
        public static void main(String[] args) {
            int[] numeri = {10, 20, 30, 40};
    
            for (int i = 0; i < numeri.length; i++) {
                System.out.println("Elemento " + i + ": " + numeri[i]);
            }
        }
    }

2. Differenze tra Frontend e Backend

Nell'ambito dello sviluppo software, distinguiamo due parti fondamentali:

  • Frontend: La parte visibile e interattiva, costruita con HTML, CSS e JavaScript. È ciò che gli utenti vedono e usano direttamente.
  • Backend: La parte nascosta, responsabile della logica applicativa, gestione dei dati e interazione con i database.

Ad esempio, quando un utente compila un modulo e clicca "Invia":

  • Il frontend raccoglie i dati dal modulo.
  • Il backend riceve questi dati, li elabora e può salvarli in un database.

Scenario di un Sistema di Login:

    Frontend: Inserimento di username e password in un modulo HTML.
    Backend: Verifica delle credenziali nel database e restituzione di una risposta.

Esempio pratico di Backend:

Un semplice esempio in Java per verificare le credenziali di accesso:

public class LoginSystem {
        public static void main(String[] args) {
            // Simulazione di un database di credenziali
            HashMap database = new HashMap<>();
            database.put("utente1", "password123");
            database.put("admin", "adminpass");
    
            Scanner scanner = new Scanner(System.in);
    
            // Input dall'utente
            System.out.print("Inserisci il tuo username: ");
            String username = scanner.nextLine();
    
            System.out.print("Inserisci la tua password: ");
            String password = scanner.nextLine();
    
            // Verifica delle credenziali
            if (database.containsKey(username) && database.get(username).equals(password)) {
                System.out.println("Accesso effettuato con successo!");
            } else {
                System.out.println("Credenziali non valide. Riprova.");
            }
        }
    }

Approfondimento sulle API

Le API (Application Programming Interfaces) sono fondamentali per la comunicazione tra frontend e backend. Scrivi un esempio di chiamata API in un'applicazione di login.

// Chiamata API dal Frontend (JavaScript)
    fetch('https://mio-sito.com/login', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({ username: 'utente1', password: 'password123' })
    })
    .then(response => response.json())
    .then(data => {
        if (data.success) {
            console.log('Login effettuato con successo');
        } else {
            console.log('Credenziali non valide');
        }
    });

3. Configurazione degli Strumenti

Un ambiente di sviluppo configurato correttamente è fondamentale per lavorare in modo efficiente.

  • Java: Installa il JDK da Oracle e configurane il percorso nel sistema.
  • Maven: Utilizza questo strumento per la gestione delle dipendenze. Configura il file pom.xml per aggiungere librerie.
  • Git: Per il controllo delle versioni. Crea un repository con git init e usa comandi come git add, git commit, git push.
  • IDE: IntelliJ IDEA o Eclipse sono strumenti ideali per sviluppare in Java con supporto per Maven.

Esempio 1: Verifica della Configurazione di Java

Un programma Java per verificare la versione e il percorso di Java installato:

public class VerificaJava {
    public static void main(String[] args) {
        System.out.println("Versione di Java installata: " + System.getProperty("java.version"));
        System.out.println("Percorso di Java: " + System.getProperty("java.home"));
    }
}

Esempio 2: Utilizzo di Maven

Comando per creare un nuovo progetto Maven:

mvn archetype:generate -DgroupId=com.example -DartifactId=my-app -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

Comandi per compilare ed eseguire un progetto Maven:

# Pulizia e compilazione del progetto
mvn clean install

# Esecuzione del progetto con Maven
mvn exec:java -Dexec.mainClass="com.example.App"

Esempio 3: Inizializzazione di un Repository Git

Comandi per configurare e inizializzare un repository Git:

# Inizializza il repository
git init

# Aggiungi tutti i file
git add .

# Commit iniziale
git commit -m "Initial commit"

# Aggiungi un remoto (es. GitHub)
git remote add origin https://github.com/username/repository.git

# Push iniziale
git push -u origin main
