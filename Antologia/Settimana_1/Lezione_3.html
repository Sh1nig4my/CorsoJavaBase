Lezione 3: Fondamenti Java

1. Introduzione alle variabili e ai tipi di dati

In Java, le variabili sono utilizzate per memorizzare dati, e ogni variabile deve essere dichiarata con un tipo specifico. Questo approccio fortemente tipizzato aiuta a prevenire errori durante l'esecuzione. I tipi di dati primitivi più utilizzati sono:

  • int: Numeri interi, ad esempio 10 o -5.
  • double: Numeri decimali, come 3.14 o -0.01.
  • char: Singoli caratteri, rappresentati da apici singoli, ad esempio 'A'.
  • boolean: Rappresenta valori di verità: true o false.

Esempio: Dichiarazione e utilizzo di variabili

public class Variabili {
    public static void main(String[] args) {
        int eta = 25;
        double salario = 45000.75;
        char grado = 'A';
        boolean isAssunto = true;

        System.out.println("Età: " + eta);
        System.out.println("Salario: " + salario);
        System.out.println("Grado: " + grado);
        System.out.println("Assunto: " + isAssunto);
    }
}

Esempio Avanzato: Utilizzo di costanti e stringhe

public class CostantiStringhe {
    public static void main(String[] args) {
        final double PI = 3.14159;
        String messaggio = "Benvenuto nel corso di Java!";

        System.out.println("Valore di PI: " + PI);
        System.out.println(messaggio);
    }
}

In questo esempio, utilizziamo una costante PI e una stringa messaggio per mostrare come lavorare con dati non primitivi e valori immutabili.

2. Gli operatori in Java

Gli operatori sono strumenti fondamentali per manipolare dati e variabili. Ecco una panoramica:

  • Aritmetici: Usati per calcoli matematici (+, -, *, /, %).
  • Relazionali: Comparano valori (>, <, >=, <=, ==, !=).
  • Logici: Combina condizioni (&&, ||, !).

Esempio: Utilizzo di operatori

public class Operatori {
    public static void main(String[] args) {
        int x = 10, y = 20;
        System.out.println("Somma: " + (x + y));
        System.out.println("Differenza: " + (x - y));
        System.out.println("Prodotto: " + (x * y));
        System.out.println("Divisione: " + (x / y));

        boolean confronto = x < y;
        System.out.println("x è minore di y: " + confronto);
    }
}

Esercizio

3. Strutture di controllo

Le strutture di controllo influenzano il flusso di esecuzione del programma, consentendo di prendere decisioni e ripetere operazioni:

  • if-else: Esegue blocchi di codice in base a condizioni.
  • switch-case: Sostituisce molteplici if per confronti diretti su un valore.
  • for: Itera per un numero specifico di volte.
  • while: Itera finché una condizione è vera.
  • do-while: Simile a while, ma garantisce almeno un'esecuzione.

Esempio: Struttura for

public class Cicli {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Iterazione: " + i);
        }
    }
}

Esempio Avanzato: Cicli Annidati

public class CicliAnnidati {
    public static void main(String[] args) {
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                System.out.println("Ciclo esterno: " + i + ", Ciclo interno: " + j);
            }
        }
    }
}

Questo esempio mostra come utilizzare cicli annidati per iterare su più livelli.

4. Esercizio Pratico: Primo Programma Java

Un classico programma Java prevede di interagire con l'utente tramite input da tastiera e output su console. Vediamo un esempio completo:

Esempio: Input e Output

import java.util.Scanner;

public class PrimoProgramma {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Inserisci il tuo nome: ");
        String nome = scanner.nextLine();

        System.out.print("Inserisci la tua età: ");
        int eta = scanner.nextInt();

        System.out.println("Ciao, " + nome + ". Hai " + eta + " anni.");
    }
}

Esercizio Pratico

Implementa un programma che calcoli la somma e la media di una serie di numeri forniti dall'utente:

  1. Chiedi all'utente di inserire il numero di elementi.
  2. Utilizza un ciclo for per leggere i numeri e calcolare la somma.
  3. Calcola la media e stampala a schermo.
