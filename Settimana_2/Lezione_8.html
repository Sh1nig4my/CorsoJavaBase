Lezione 8: Date and Time API, Parallel Streams e Collector API

Benvenuti alla lezione dedicata a tre fondamentali innovazioni introdotte con Java 8: la nuova API per la gestione di date e orari (Date and Time API), l'uso degli stream paralleli per sfruttare al meglio le capacità delle moderne CPU multi-core, e l'API Collector per semplificare operazioni complesse di aggregazione e trasformazione dei dati.

Queste funzionalità non solo migliorano le performance e la leggibilità del codice, ma promuovono anche pratiche di programmazione più robuste e moderne. Attraverso spiegazioni dettagliate ed esempi pratici, acquisiremo una comprensione approfondita di questi strumenti indispensabili.

Date and Time API

La Date and Time API, introdotta con il pacchetto java.time, è stata progettata per risolvere i limiti delle vecchie classi come java.util.Date e Calendar. Tra i vantaggi principali troviamo:

  • Immutabilità: le classi come LocalDate, LocalTime e LocalDateTime sono immutabili, riducendo il rischio di errori dovuti a modifiche inaspettate.
  • Chiarezza: nomi di metodi e classi intuitivi rendono il codice più leggibile e comprensibile.
  • Supporto a fusi orari: classi come ZonedDateTime semplificano la gestione di date e orari con fusi orari differenti.

Esempio pratico: Utilizzo di LocalDate, LocalTime e LocalDateTime

public class DateAndTimeExample {
    public static void main(String[] args) {
        // Ottenere la data, l'orario e il timestamp corrente
        LocalDate today = LocalDate.now();
        LocalTime now = LocalTime.now();
        LocalDateTime currentDateTime = LocalDateTime.now();

        System.out.println("Data odierna: " + today);
        System.out.println("Orario attuale: " + now);
        System.out.println("Data e ora correnti: " + currentDateTime);
    }
}

Esempio pratico: Calcoli con date

public class DateCalculations {
    public static void main(String[] args) {
        LocalDate birthDate = LocalDate.of(1990, 5, 15);
        LocalDate today = LocalDate.now();

        Period age = Period.between(birthDate, today);
        System.out.println("Età: " + age.getYears() + " anni, " + age.getMonths() + " mesi, " + age.getDays() + " giorni.");
    }
}

Parallel Streams

Con i parallel streams, Java 8 ha introdotto un modo semplice per eseguire operazioni su collezioni sfruttando il parallelismo. Questo approccio si basa sul framework Fork/Join, che suddivide il lavoro in task più piccoli, gestiti in parallelo da più thread.

È importante notare che, sebbene il parallelismo possa migliorare le performance, il suo utilizzo richiede attenzione. Operazioni non thread-safe o la sincronizzazione eccessiva possono annullare i vantaggi offerti.

Esempio pratico: Differenza tra stream sequenziali e paralleli

public class ParallelStreamsExample {
    public static void main(String[] args) {
        List items = Arrays.asList("Apple", "Banana", "Cherry", "Date", "Elderberry");

        System.out.println("Elaborazione sequenziale:");
        items.stream().forEach(System.out::println);

        System.out.println("\nElaborazione parallela:");
        items.parallelStream().forEach(System.out::println);
    }
}

Esempio pratico: Somma parallela di numeri

public class ParallelSumExample {
    public static void main(String[] args) {
        int sum = IntStream.range(1, 1000).parallel().sum();
        System.out.println("Somma parallela: " + sum);
    }
}

Collector API

L'API Collector di Java 8 semplifica la raccolta e l'aggregazione dei dati da uno stream. Permette di creare mappe, liste e set con pochi passaggi, oltre a supportare operazioni personalizzate come la riduzione o il raggruppamento.

La flessibilità di questa API la rende adatta a molti scenari, dalla trasformazione di dati alla creazione di strutture dati complesse.

Esempio pratico: Raggruppare elementi

public class CollectorExample {
    public static void main(String[] args) {
        List items = Arrays.asList("Apple", "Banana", "Cherry", "Apple");

        Map itemCount = items.stream().collect(Collectors.groupingBy(
                item -> item,
                Collectors.counting()
        ));

        itemCount.forEach((item, count) -> System.out.println(item + ": " + count));
    }
}

Esempio pratico: Calcolo della media

public class AverageExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(10, 20, 30, 40, 50);

        double average = numbers.stream()
                                .collect(Collectors.averagingInt(num -> num));

        System.out.println("Media: " + average);
    }
}
