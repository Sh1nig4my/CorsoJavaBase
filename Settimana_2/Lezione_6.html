Lezione 6: Lambda Expressions, Functional Interfaces e Default Methods

Introduzione

Con l'introduzione di Java 8, il linguaggio ha subito una rivoluzione che ha reso possibile adottare uno stile di programmazione più funzionale. La sesta lezione esplorerà tre concetti fondamentali:

  • Lambda Expressions: un modo conciso per implementare metodi astratti.
  • Functional Interfaces: interfacce con un singolo metodo astratto che fungono da target per le Lambda Expressions.
  • Default Methods: la possibilità di definire metodi con implementazione predefinita nelle interfacce.

Lambda Expressions

Le Lambda Expressions sono un meccanismo per implementare metodi in modo più compatto. Permettono di eliminare la verbosità associata alle classi anonime, migliorando leggibilità e mantenibilità del codice.

Sintassi

(parametri) -> espressione
(parametri) -> {
    // corpo della funzione
}

Ad esempio:

interface Operazione {
    int calcola(int a, int b);
}

public class LambdaExample {
    public static void main(String[] args) {
        Operazione somma = (a, b) -> a + b;
        Operazione moltiplicazione = (a, b) -> a * b;

        System.out.println("Somma: " + somma.calcola(5, 3));
        System.out.println("Moltiplicazione: " + moltiplicazione.calcola(5, 3));
    }
}

In questo esempio, usiamo due implementazioni lambda per eseguire operazioni matematiche.

Functional Interfaces

Le Functional Interfaces sono interfacce che definiscono un solo metodo astratto. Sono annotate con @FunctionalInterface per garantire questa caratteristica.

Esempio di Functional Interface

@FunctionalInterface
interface Saluto {
    void saluta(String nome);
}

public class FunctionalInterfaceExample {
    public static void main(String[] args) {
        Saluto salutoFormale = nome -> System.out.println("Buongiorno, " + nome + "!");
        Saluto salutoInformale = nome -> System.out.println("Ciao, " + nome + "!");

        salutoFormale.saluta("Luca");
        salutoInformale.saluta("Anna");
    }
}

Questo esempio dimostra due implementazioni diverse di una functional interface per i saluti.

Default Methods

I Default Methods permettono di aggiungere metodi con implementazione predefinita nelle interfacce. Ciò consente di estendere le funzionalità di un'interfaccia senza interrompere il codice esistente.

Esempio di Default Method

interface Calcolatrice {
    default int sottrai(int a, int b) {
        return a - b;
    }

    int moltiplica(int a, int b);
}

public class DefaultMethodExample implements Calcolatrice {
    public int moltiplica(int a, int b) {
        return a * b;
    }

    public static void main(String[] args) {
        DefaultMethodExample calcolatrice = new DefaultMethodExample();
        System.out.println("Sottrazione: " + calcolatrice.sottrai(10, 5));
        System.out.println("Moltiplicazione: " + calcolatrice.moltiplica(10, 5));
    }
}
