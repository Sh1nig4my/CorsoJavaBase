Lezione Giorno 7: Stream API e Optional

Introduzione

Durante questa lezione esploreremo due delle funzionalità più potenti introdotte in Java 8: la Stream API e la classe Optional. Questi strumenti rappresentano un cambiamento significativo nel modo in cui i programmatori Java affrontano la manipolazione dei dati e la gestione dei valori null.

Stream API

La Stream API permette di lavorare con sequenze di dati in modo dichiarativo e funzionale. Le operazioni su uno stream possono essere classificate in:

  • Operazioni intermedie: trasformano uno stream in un altro (es. filter, map).
  • Operazioni terminali: producono un risultato finale o un effetto collaterale (es. collect, forEach).

Un aspetto fondamentale degli Stream è che lavorano in modo lazy, cioè le operazioni intermedie non vengono eseguite finché non viene chiamata un'operazione terminale. Questo approccio migliora l'efficienza, specialmente su grandi volumi di dati.

Esempio Pratico 1

Filtrare una lista di numeri pari e sommarli:

public class StreamExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(1, 2, 3, 4, 5, 6);

        int sum = numbers.stream()
                         .filter(n -> n % 2 == 0)
                         .mapToInt(Integer::intValue)
                         .sum();

        System.out.println("Somma dei numeri pari: " + sum);
    }
}

Esempio Pratico 2

Convertire una lista di stringhe in maiuscolo:

public class StreamUppercaseExample {
    public static void main(String[] args) {
        List words = Arrays.asList("java", "stream", "api");

        List uppercasedWords = words.stream()
                                            .map(String::toUpperCase)
                                            .collect(Collectors.toList());

        System.out.println("Parole in maiuscolo: " + uppercasedWords);
    }
}

Esempio Pratico 3

Calcolare la lunghezza media delle parole:

public class StreamAverageExample {
    public static void main(String[] args) {
        List words = Arrays.asList("hello", "world", "java", "stream");

        double averageLength = words.stream()
                                    .mapToInt(String::length)
                                    .average()
                                    .orElse(0.0);

        System.out.println("Lunghezza media delle parole: " + averageLength);
    }
}

Optional

La classe Optional è stata introdotta per evitare il rischio di NullPointerException. Essa incapsula un valore che può essere presente o assente e offre metodi funzionali per gestire questi casi.

Optional semplifica l'esecuzione di operazioni sicure su valori opzionali, come controllare la presenza del valore, trasformarlo o fornire un valore predefinito in sua assenza.

Esempio Pratico 1

Utilizzo di Optional per gestire un valore potenzialmente nullo:

public class OptionalExample {
    public static void main(String[] args) {
        Optional optionalValue = Optional.ofNullable(getValue());

        optionalValue.ifPresentOrElse(
            value -> System.out.println("Valore presente: " + value),
            () -> System.out.println("Valore assente")
        );
    }

    public static String getValue() {
        return null; // Prova con "Hello, World!" per cambiare il risultato
    }
}

Esempio Pratico 2

Fornire un valore predefinito con orElse:

public class OptionalDefaultExample {
    public static void main(String[] args) {
        String value = Optional.ofNullable(null)
                               .orElse("Valore di default");

        System.out.println("Risultato: " + value);
    }
}

Esempio Pratico 3

Utilizzare map per trasformare un valore presente:

public class OptionalMapExample {
    public static void main(String[] args) {
        Optional optionalValue = Optional.of("java");

        optionalValue.map(String::toUpperCase)
                     .ifPresent(value -> System.out.println("Valore trasformato: " + value));
    }
}
