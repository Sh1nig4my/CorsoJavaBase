Esercizio Giorno 5: Utilizzo di List, Map e Salvataggio su File JSON

Obiettivi

In questa esercitazione, utilizzerai le interfacce List e Map della libreria Java Collections per gestire dati dinamici. Inoltre, implementerai il salvataggio e il caricamento di dati in formato JSON usando librerie come Gson o Jackson.

Descrizione dell'Esercizio

Scrivi un programma in Java che:

  1. Legga una lista di numeri interi forniti dall'utente e li memorizzi in una List (ad esempio, ArrayList).
  2. Classifichi i numeri come "Pari" o "Dispari" utilizzando una Map con due chiavi: "Pari" e "Dispari".
  3. Calcoli e stampi il numero più grande, il numero più piccolo e la media dei numeri.
  4. Salvi i numeri originali, la classificazione e le statistiche in un file JSON chiamato risultati.json.
  5. Consenta di caricare dati da un file JSON esistente e di continuare l'analisi.

Il programma deve gestire input non validi e chiedere nuovamente i dati finché non vengono forniti numeri validi.

Esempio di Input e Output

Input:

Inserisci i numeri (separati da spazio): 10 20 15 7 9 3

Output Console:

Numero più grande: 20
Numero più piccolo: 3
Media: 10.67
Numeri pari: 10, 20
Numeri dispari: 15, 7, 9, 3
Dati salvati con successo in risultati.json

Contenuto del File JSON:

{
  "numeriOriginali": [10, 20, 15, 7, 9, 3],
  "analisi": {
    "numeroPiuGrande": 20,
    "numeroPiuPiccolo": 3,
    "media": 10.67
  },
  "classificazione": {
    "Pari": [10, 20],
    "Dispari": [15, 7, 9, 3]
  }
}

Esempio di Creazione di un File JSON:

try (FileWriter writer = new FileWriter("risultati.json")) {
    Gson gson = new Gson();
    gson.toJson(datiAnalizzati, writer);
    System.out.println("File JSON creato con successo!");
} catch (IOException e) {
    e.printStackTrace();
}

Esempio di Modifica di un File JSON:

try (FileReader reader = new FileReader("risultati.json")) {
    Gson gson = new Gson();
    Map dati = gson.fromJson(reader, Map.class);
    List nuoviNumeri = Arrays.asList(25, 30);
    ((List) dati.get("numeriOriginali")).addAll(nuoviNumeri);

    try (FileWriter writer = new FileWriter("risultati.json")) {
        gson.toJson(dati, writer);
        System.out.println("File JSON aggiornato con nuovi dati!");
    }
} catch (IOException e) {
    e.printStackTrace();
}

Esempio di Eliminazione di un File JSON:

File file = new File("risultati.json");
if (file.delete()) {
    System.out.println("File JSON eliminato con successo!");
} else {
    System.out.println("Impossibile eliminare il file.");
}

Requisiti

  • Usa una List per memorizzare i numeri forniti dall'utente.
  • Utilizza una Map per classificare i numeri in "Pari" e "Dispari".
  • Adotta una libreria come Gson o Jackson per il salvataggio e il caricamento dei dati in formato JSON.
  • Assicurati di validare gli input dell'utente.
  • Documenta il codice per renderlo comprensibile.

Per gli Studenti Avanzati

Prova a implementare le seguenti funzionalità aggiuntive:

  • Permetti all'utente di aggiornare i dati caricati da un file JSON e salvarli nuovamente.
  • Aggiungi una funzione per esportare i dati in formato CSV.
  • Offri la possibilità di filtrare i numeri caricati in base a un intervallo specificato dall'utente.
  • Implementa un menu interattivo per scegliere le diverse operazioni (inserimento, salvataggio, caricamento, analisi, esportazione).
