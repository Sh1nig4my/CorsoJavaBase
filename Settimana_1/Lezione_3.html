Lezione 3 - Fondamenti Java

In questa lezione ci concentreremo sui fondamenti del linguaggio Java, essenziali per costruire solide basi di programmazione. Tratteremo variabili, tipi di dati, operatori e strutture di controllo, concludendo con la realizzazione di un primo programma Java per la gestione di input e output da console.

Argomenti Trattati

  • Introduzione alle variabili e ai tipi di dati: Concetti base, tipi primitivi e oggetti in Java.
  • Operatori: Aritmetici, logici, relazionali e operatori speciali.
  • Strutture di controllo: If-else, switch-case, cicli for, while e do-while.
  • Primo programma Java: Creazione di un programma per la gestione dell'input/output.

Linguaggio Java

Variabili e Tipi di Dati

Le variabili sono contenitori per memorizzare dati. In Java, ogni variabile deve avere un tipo definito. I tipi primitivi più comuni includono:

  • int: numeri interi.
  • double: numeri in virgola mobile.
  • char: un singolo carattere.
  • boolean: valori true o false.
Esempio: 
public class Variabili {
    public static void main(String[] args) {
        int numero = 10;
        double prezzo = 19.99;
        char iniziale = 'J';
        boolean isOnline = true;

        System.out.println("Numero: " + numero);
        System.out.println("Prezzo: " + prezzo);
        System.out.println("Iniziale: " + iniziale);
        System.out.println("Online: " + isOnline);
    }
}

Operatori

Gli operatori permettono di eseguire operazioni su variabili e valori. I principali sono:

  • Aritmetici: + - * / %
  • Relazionali: > < >= <= == !=
  • Logici: && || !
Esempio: 
public class Operatori {
    public static void main(String[] args) {
        int a = 10, b = 20;
        System.out.println("Somma: " + (a + b));
        System.out.println("Maggiore: " + (a > b));
        System.out.println("E logico: " + (a > 5 && b > 15));
    }
}

Strutture di Controllo

Le strutture di controllo determinano il flusso del programma. Esempi includono:

  • if-else: per scelte condizionali.
  • switch-case: per selezioni multiple.
  • for, while, do-while: per cicli iterativi.
Esempio: 
public class Controllo {
    public static void main(String[] args) {
        int voto = 85;

        if (voto >= 90) {
            System.out.println("Ottimo!");
        } else if (voto >= 70) {
            System.out.println("Buono!");
        } else {
            System.out.println("Da migliorare.");
        }
    }
}

Primo Programma Java

Un programma Java inizia con un metodo main. Qui creiamo un programma per ricevere input e mostrare output.

Esempio: 
import java.util.Scanner;

public class PrimoProgramma {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Inserisci il tuo nome: ");
        String nome = scanner.nextLine();

        System.out.println("Ciao, " + nome + "!");
    }
}
